# random_user_agent
A library for generating a random User-Agent for parsing websites from an embedded file.

# Installation

To install the library into your project, you need to run the following command:
```bash
pip install "git+https://github.com/vvetta/random_user_agent.git@main"
```

# File with User-Agent

The embedded file is taken from this [gist](https://gist.github.com/pzb/b4b6f57144aea7827ae4)!
